# M46-M50 - Dynamic arrays, bool/string arrays, local arrays and array params

## Summary

This block expanded arrays from fixed global int arrays into a broader runtime data model.

## What happened

- M46 added dynamic runtime-int indexing with bounds-checked dispatch/trap behavior.
- M47 added runtime bool arrays.
- M48 added runtime string arrays and array length helpers.
- M49 added local runtime arrays inside functions.
- M50 added fixed-size arrays as function params with copy-in/copy-back behavior.

## Result

Arrays became usable in function scope and as parameters, while still intentionally avoiding array returns, push/pop, slices and dynamic resizing.
