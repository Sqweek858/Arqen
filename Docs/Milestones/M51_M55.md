# M51-M55 - Array utilities and records

## Summary

This block added utility operations around arrays and introduced records as grouped runtime data.

## What happened

- M51 added array fill/copy utilities for int, bool and string arrays.
- M52 introduced runtime records with int/bool/string fields.
- M53 added local records and record params with copy-in/copy-back behavior.
- M54 added record arrays with static and dynamic index field access.
- M55 added record copy/reset and record array copy/reset utilities.

## Result

Arqen gained a simple structured data model, still lowered through the existing slot/action approach rather than packed memory structs.
