# M41-M45 - Typed function returns, params, locals, call graph and int arrays

## Summary

This block turned functions into useful typed units and started array support.

## What happened

- M41 added typed function returns for int, bool and string.
- M42 added typed function parameters for int, bool and string.
- M43 added local runtime scope inside functions.
- M44 hardened function-to-function calls, nested typed assignments, define-before-call and recursion/cycle rejection.
- M45 introduced fixed-size runtime int arrays with static indexing and compile-time bounds validation.

## Result

The language gained a usable function model and the first fixed-size runtime array model.
