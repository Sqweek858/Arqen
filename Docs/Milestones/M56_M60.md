# M56-M60 - Enums, switch and enum integration

## Summary

This block added enums and connected them to functions, records and arrays.

## What happened

- M56 introduced enum definitions, runtime enum slots, set/copy and runtime enum conditions.
- M57 added runtime switch over enum values.
- M58 added runtime switch over int values and hardened switch validation.
- M59 added enum params and enum returns.
- M60 added enum record fields and enum arrays.

## Result

Enums became integrated with the typed runtime model while remaining int-backed internally.
