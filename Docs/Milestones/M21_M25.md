# M21-M25 - Shader pipeline and first scene/runtime graphics work

## Summary

This block continued the DX12 branch from a clear-window smoke test toward a small scene/runtime path.

## What happened

- Shader pipeline metadata and HLSL/runtime integration were documented and validated.
- A triangle smoke test and standalone frame loop were added.
- Constant color animation and generated DX12 metadata became part of the pipeline.
- Scene/object concepts started forming, including early crystal scene experiments.

## Result

The DX12 branch became capable of repeated frame output and simple scene construction rather than only a one-frame clear operation.
