# M36-M40 - Runtime conditions, string data and function foundation

## Summary

This block strengthened runtime control and introduced functions.

## What happened

- M36 added runtime string comparisons, bool not/toggle and mixed condition polish.
- M37 added runtime string/data utilities: contains, case-insensitive equality/inequality, concat, substring and parse-int.
- M38 audited the function/runtime foundation and registry coverage.
- M39 introduced function IR/runtime call-return foundation without params or return values.
- M40 added void return and early return support.

## Result

Functions became part of the runtime model, first as a stable call/return foundation before adding values, parameters and local state.
