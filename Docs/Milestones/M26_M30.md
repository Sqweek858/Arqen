# M26-M30 - Camera, input, object selection and UI overlay bridge

## Summary

This block pushed the experimental DX12 branch toward an editor/game-like viewport and UI bridge.

## What happened

- Runtime scene controls and camera interaction were expanded.
- Perspective/depth support improved the visual pipeline.
- Window-style boxes, keyboard/mouse input and peripheral handling were added.
- Rotation/light demos, UE-style viewport navigation and object selection/rotation were introduced.
- M30 started UI overlay controls and layout bridge work.

## Result

Arqen's DX12 branch became a practical testbed for viewport-style interaction and UI metadata, though still experimental and not the main production target.
